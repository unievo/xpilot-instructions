# mxpy config set

Sets a configuration value.

## Usage

```bash
mxpy config set [name] [value]
```

## Parameters

- `name` (string)  
  The name of the configuration entry.
- `value` (string)  
  The new value to set.
- `-h, --help`  
  Show help message and exit.

## Example

```bash
mxpy config set proxy https://api.multiversx.com
